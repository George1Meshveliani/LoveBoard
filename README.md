# LoveBoard

A simple drawing app built with Love2D.

## Downloads

- **Windows**: Download `LoveBoard.exe` and run directly.  
- **Linux / macOS**: Download `LoveBoard.love` and run with Love2D installed:
  ```bash
  love LoveBoard.love
